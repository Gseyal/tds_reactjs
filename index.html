def calculate_pnl(trades, benchmark_date):
    """
    Calculates Profit & Loss (PnL) for a list of trades
    against a given benchmark.
    """
    pnl = sum(t.return_pct for t in trades if t.date < benchmark_date)
    return pnl

üìê Capital Asset Pricing Model (CAPM)

We use CAPM to determine the theoretical required rate of return for our strategic investments:

$$ E(R_i) = R_f + \beta_i (E(R_m) - R_f) $$
  • $E(R_i)$: Expected return on asset $i$
  • $R_f$: Risk-free rate
  • $\beta_i$: Beta of asset $i$ (Systematic risk)
  • $E(R_m)$: Expected market return
